## Agenda

### 1. Welcome + dry-run framing (5 min)

- Purpose of this meeting and its relationship to RFC-0026 RV5-CA07-CCM
  Pathway 2.
- Grace-period status (2026-06-30 → 2026-12-31).
- Explicit statement that this is a dry run; nothing here is final
  compliance evidence, but every artifact produced is real.

### 2. POA&M delta since last meeting (15 min)

- Walk-through of `exports/conmon/conmon-poam.csv` (July 2026 run).
- What opened, what closed, what changed severity since the baseline
  established at the effective date.
- First exercise of the aggregator's output reading front-to-back.

### 3. Dashboard review: RFC-0026 posture per adapter (10 min)

- `exports/conmon/conmon-aggregate-summary.json` walkthrough.
- Per adapter: requirement / pathway / last run / finding counts / SLA
  state.
- Explicit note for any CA-7-tagged adapter still in `status: reserved`
  — what that means for coverage and when it promotes.

### 4. New critical findings + 72-hour SLA state (10 min)

- Any finding that tripped the self-imposed 72-hour critical SLA
  (ADR-043 D2). For the dry run, expected state is zero breaches; if
  any, walk through the `conmon-cure` governance issue opened by
  `conmon-aggregate.yml`.
- Review the 45-day internal cure window (per
  `docs/docs/conmon-corrective-action-playbook.qmd`).

### 5. Upstream change summary (5 min)

- ScubaGear pin bumps landed since the effective date.
- RFC-0026 ratification status (still PROPOSED? ratified?).
- VDR / CCM Balance Improvement Release publication status (gating
  the Pathway-1 migration per UIAO_132 §2.4 and §3.4).
- Relevant CISA BOD / emergency directive activity.

### 6. Significant Change Request review (5 min)

- Any SCR filed or anticipated for Q3 2026.
- Template: source the process from the FedRAMP ConMon Playbook local
  copy under `src/uiao/canon/compliance/reference/fedramp-conmon-playbook/`.

### 7. Agency Q&A / open floor (10 min)

- Standing fixture per `docs/docs/conmon/index.qmd` §Standing agenda.
- Questions on evidence consumption via OSCAL feeds.
